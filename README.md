# Эксперимент по ансамблированию моделей для прогнозирования временных рядов (M4)

## Суть эксперимента
Цель работы — проверка гипотезы: **ансамблирование простых разнородных моделей даёт качество лучше и/или стабильнее, чем одна «лучшая» модель**.

Эксперимент проводился на подвыборке из **200 временных рядов** для каждой из трёх частот (годовые, квартальные, месячные) из набора данных [M4](https://forecastingdata.org/).  
В качестве базовых моделей использовались:
- **PatchTST** – современная нейросетевая модель (глобальное обучение);
- **Theta** – классический статистический метод (локальное обучение).

Были реализованы следующие подходы:
- простые бейзлайны (Naive, SeasonalNaive, AutoETS, Theta);
- глобальные модели CatBoost и PatchTST;
- простое усреднение прогнозов;
- стекинг с мета-моделью CatBoost, включающий статические признаки рядов и корректную кросс-валидацию.

**Основной результат:** стекинг позволил снизить ошибку sMAPE на **38%** относительно лучшей индивидуальной модели и уменьшить разброс ошибок почти вдвое, что подтверждает гипотезу.

## Инструкция по запуску

### 1. Установка зависимостей
Проект использует Python 3.9+. Установите необходимые библиотеки:
```
pip install -r requirements.txt
```
### 2. Конфигурация
Все основные параметры (пути к данным, количество рядов, настройки моделей) вынесены в файл config.yaml.
При необходимости отредактируйте его перед запуском.

3. Запуск эксперимента
Для полного воспроизведения всех шагов выполните:

```
cd src
python main.py
```

Скрипт последовательно:

загрузит данные M4 и сформирует выборку из 200 рядов на частоту;

создаст валидационные окна;

рассчитает бейзлайны;

обучит глобальные модели (CatBoost и PatchTST);

построит простые ансамбли и выполнит стекинг с кросс-валидацией.
